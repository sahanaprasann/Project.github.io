MAJOR PROJECT

Smart Car Parking System Using IOT

Description: Description: Smart parking system is an integrated system to organize the cars in parking slot. The main aim of this project is to digitize the parking system using blynk application which is connected to the IOT cloud. The users can check the availability of the parking slot using IOT cloud. The users can register their details with the authorized person and get the device installed to their cars. By pressing the button which is installed in the car . The gate is opened by the vehicle Authorized Owner when the vehicle is front of the gate by Using Zigbee Technology. Technology: code is implemented in embedded C.
